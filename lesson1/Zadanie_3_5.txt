ЗАДАНИЕ 3
Объяснить, как работает данный код:

$a = 5;
$b = '05';


var_dump($a == $b) Почему true?
 true потому здесь происходит сравнение переменных без учёта типов данных и строка преобразуется в число.

var_dump((int)'012345');  Почему 12345?
Строка преобразовывается в целочисленный тип данных. Вывод: int(12345) - тип данных и значение.

var_dump((float)123.0 === (int)123.0); Почему false?
При тождественном сравнении сравнивается ещё и тип данных. Дробный тип и целочисленный не равны друг другу, потому  false.

var_dump((int)0 === (int)'hello, world'); Почему true?
В данном случае мы строку приводим к целочисленному типу данных. Если строка содержит число или ведущую числовую последовательность, 
тогда она будет преобразована в соответствующее целое число, в противном случае она преобразуется в ноль. 0 === 0 true
